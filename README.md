# 📖 Project Overview

The **Parking Management System** is a desktop-based application developed using **C# Windows Forms** and **SQL Server**.  
This system helps automate vehicle parking management by handling:

- Vehicle Entry
- Vehicle Exit
- Parking Slot Monitoring
- Payment Calculation
- Parking Record Management
- Search & Delete Operations

The project was developed as an academic project for **Object Oriented Programming 2 (OOP2)**.
